# Contributing to ScholrZA

Thanks for your interest in contributing! Here's how to get involved.

## Reporting Bugs

Open an issue and include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your device and browser

## Suggesting Features

Open an issue with the label `enhancement` and describe your idea clearly.

## Pull Requests

1. Fork the repo
2. Create a new branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test everything works
5. Submit a Pull Request with a clear description

## Code Style

- Keep it clean and readable
- Comment anything complex
- Test on mobile (most users are on phones)

## Questions?

Open an issue or reach out via the live site at [scholrza.pages.dev](https://scholrza.pages.dev).
