## [2.0.0] - 2026-03

### Added
- 162+ RSS feeds with automatic category detection
- AI Summary and Text-to-Speech on news articles
- Google AdSense integration (display + in-article ads)
- Weekly email newsletter system via Resend
- Signed Android APK via GitHub Actions (PWABuilder TWA)
- Google Search Console verification
- Service worker upgraded to scholrza-v3 (stops caching API/HTML)

### Fixed
- AI chat broken due to intentional proxy error — removed and fixed
- Service worker no longer caches dynamic content
