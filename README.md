## What is ScholrZA?

ScholrZA is a free, student-built platform designed specifically for South African university students. Built by a student, for students — covering everything you need to survive and succeed at varsity.

## Features

- 📋 **NSFAS** — Funding info, application guides, and updates
- 💰 **Bursaries** — Find and apply for bursaries across South Africa
- 📄 **Past Papers** — Exam papers from 2015–2024, all grades
- 🏠 **Accommodation** — Student accommodation listings and tips
- 💼 **Jobs** — Part-time and graduate job opportunities
- 📰 **News** — Live South African student news from 162+ RSS feeds
- 🤖 **AI Assistant** — Built-in AI chat to help with student questions
- 🔊 **Text-to-Speech** — Listen to news articles hands-free
- 📱 **PWA + APK** — Installable as an app on Android

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend/Functions:** Cloudflare Pages Functions
- **Database:** Supabase
- **Auth:** Google OAuth + Supabase
- **Hosting:** Cloudflare Pages
- **Notifications:** Resend (email newsletter)
- **Ads:** Google AdSense
